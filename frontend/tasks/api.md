Реализовать API для взаимодействия с сервером согласено документу `/docs/ENDPOINTS.md`

- Ипользовать Zod для валидации данных
- Для каждого эндпоинта реализовать функцию и поместить в отдельный файл в папке `/src/shared/api`
- Функция должна принимать валидные данные и возвращать промис с данными от сервера
